Vector Space Model

Vector space model or term vector model is an algebraic model for representing text documents (and any objects, in general) as vectors of identifiers (such as index terms). It is used in information filtering, information retrieval, indexing and relevancy ranking

Advantages Over Boolean Retrieval Model

The vector space model has the following advantages over the Standard Boolean model:

  • Simple model based on linear algebra
  • Term weights not binary
  • Allows computing a continuous degree of similarity between queries and documents
  • Allows ranking documents according to their possible relevance
  • Allows partial matching
